# PHOTONHUNT
## ゲーム概要
影に変身して、敵を倒しながらゴールを目指す2Dアクションゲーム。

## 操作方法
### コントローラ
左スティック：移動

Aボタン：変身

Bボタン（長押し->離す）：攻撃　※変身中のみ可能

### キーボード
WASD：移動

Q：変身

E（長押し->離す）：攻撃　※変身中のみ可能

## 使用技術
・C++
・DirectX11

## 工夫した点
影の描画が黒のピクセルテクスチャのみで、4角の描画座標を計算し滑らかな影の描画を実装。

壁の後ろにプレイヤーがいる際に該当の壁を透過。

エネミーがプレイヤーを追跡する際に壁に引っかからない処理を実装。
